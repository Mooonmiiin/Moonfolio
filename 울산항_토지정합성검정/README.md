# 울산항 육상항만구역 데이터 정합성 점검

## 📌 개요
울산항 육상항만구역 토지 데이터는 여러 기관에서 분산 관리되며,
소재지, 면적, 지목 기준 불일치 문제가 발생하고 있습니다.

본 프로젝트는 토지이음 데이터를 기반으로 기준 데이터를 구축하고,
정부24 데이터를 활용한 이중 검증을 통해 데이터 정합성을 점검합니다.

---

## 🔄 파이프라인

01. 소재지 정리  
02. 토지이음 크롤링  
03. 데이터 병합  
04. 정부24 조회 대상 생성  
05. 정부24 크롤링  
06. 기준 데이터 생성  
07. 정합성 검증  

---

## ⚙️ 주요 특징

- 토지이음 크롤링을 통한 기준 데이터 구축
- 정부24 데이터를 활용한 이중 검증
- 면적 및 지목 기준 데이터 불일치 탐지
- 단계별 중간 데이터를 저장하여 재실행 가능

---

## 📂 폴더 구조

각 단계별로 input/output 데이터를 함께 구성하여  
부분 실행이 가능하도록 설계했습니다.

---

## 📊 결과

- 소재지 점검: 35건 (말소 17건 / 비말소 18건)
- 면적 불일치: 73건
- 지목 불일치: 23건
